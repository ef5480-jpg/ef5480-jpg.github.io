Listicle of Fabric Modding Things

  • Fabric Wiki - Fairly useful documentation of the Fabric loader
  • Linkie - Can view and convert mappings
  • Modding By Kaupenjoe - Extremely useful tutorials for both java and modding
  • Blockbench Style Guide - Useful for replicating the Minecraft style of pixel art and models
  • Pixabay - Lots of free sounds to use
  • Audacity - A way to edit sounds
